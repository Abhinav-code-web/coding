# 📊 Retail Sales Performance & Predictive Analytics Dashboard

> An interactive **Power BI** dashboard designed to analyze retail
> sales, profitability, regional performance, and future sales trends
> using predictive analytics.

------------------------------------------------------------------------

## 📌 Project Overview

This project provides a comprehensive view of retail sales performance
through interactive Power BI dashboards. It enables users to monitor key
business KPIs, analyze sales trends, compare regional performance, and
explore category-level insights to support data-driven decision-making.

------------------------------------------------------------------------

## 🎯 Objectives

-   Monitor overall sales performance.
-   Track Total Sales, Profit, Quantity, and Profit Margin.
-   Analyze sales by region and category.
-   Compare sales across different years.
-   Forecast future sales using predictive analytics.
