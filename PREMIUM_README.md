## 📌 Project Overview

This project demonstrates an end-to-end **Retail Sales Analytics
Dashboard** developed in **Power BI**. The dashboard transforms raw
retail sales data into interactive visualizations that help
decision-makers monitor KPIs, identify trends, evaluate regional
performance, and forecast future sales.

------------------------------------------------------------------------

## 🎯 Business Objectives

-   Monitor Sales, Profit, Quantity Sold, and Profit Margin.
-   Compare current and historical sales performance.
-   Analyze sales across regions, categories, and sub-categories.
-   Identify top-performing products.
-   Support business decisions using predictive analytics.
